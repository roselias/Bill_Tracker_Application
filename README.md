Project created for UHD's Human-Computer Interaction Summer Course. Written in Java and implementing SQLite. 
The purpose of this project is to create an mobile app for users who would like to track the due dates and payments made of their bills and subscriptions. 
At the time of presentation on 08/2024, the app allowed users to add bills to keep track of them in the app.
